#Bodeguitas WONG

El presente proyecto es brindar una solución integral para registrar los pedidos de los vendedores, así como consultar en tiempo real la mercadería disponible para ofertar a los clientes (promociones)

# User Story
Iniciar sesión:

	Permite registrarse como usuario.

Registro de Clientes:

	Registrar nuevos clientes, previa verificación en SUNAT e INFOCORP

Registro de Pedido:

	Registrar pedidos, verificando el cliente y productos en promoción.

Promociones:

	Registro y visualización de productos en promoción.

Mensajería:

	El vendedor confirma el pedido y se envía  la información al almacén para su atención
